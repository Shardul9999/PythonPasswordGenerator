# 🔐 Password Manager (Tkinter)

A simple desktop **Password Manager** built using Python and Tkinter.
It allows users to generate secure passwords and save login credentials locally.

---

## 🚀 Features

* 🔑 Generate strong random passwords
* 📏 Custom password length (user-defined)
* 🧩 Includes letters, numbers, and symbols
* 💾 Save credentials (Website, Email, Password)
* 🖥️ Simple and clean GUI using Tkinter
* ⚠️ Input validation with pop-up alerts

---

## 🛠️ Tech Stack

* Python 🐍
* Tkinter (GUI)
* Random module

## 💡 How It Works

* Enter website and email/username
* Input desired password length
* Click **Generate Password**
* Click **Add** to save credentials

Passwords are stored locally in a `data.txt` file.

---

## ⚠️ Note

* This project stores data in plain text (not encrypted)
